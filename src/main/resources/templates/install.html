Open the ISO file in a virtual machine like these

Depending of the operating system you can use the following applications or commands

Linux

Console
To write your ISO file run the following command (it may take a while)
sudo dd if=/path/file.iso of=/dev/device bs=16M status=progress oflag=sync
For search the /dev/device destination name insert the USB stick and open the disk utility in the system tools folder of applications screen
Select the device and in the device field (Dispositivo in spanish) you have the destination, in this case /dev/sda1. The destination is that without the number, example: /dev/sda
