Installation
The Debian installer makes easy the installation throught a few steps. The list of steps are the following
Note: These are the most common steps in the installation. Depending on the selected options, some other steps may be displayed.

Choose the language to be used for the installation process. The selected language will also be the default language for the installed system.

The selected location will be used to set your time zone and also for example to help select the system locale.

Select your keyboard keymap distribution.

The hostname is a single word that identifies your system to the network. If you don't know what your hostname should be, consult your network administrator. If you are setting up your own home network, you can make something up here.

This is an optional value. The domain name is the part of your Internet address to the right of your host name. It is often something that ends in .com, .net, .edu, or .org. If you are setting up a home network, you can make something up, but make sure you use the same domain name on all your computers.

Please enter the real name of this user. This information will be used for instance as default origin for emails sent by this user as well as any program which displays or uses the user's real name. Your full name is a reasonable choice.

Select a username for the new account. Your first name is a reasonable choice. The username should start with a lower-case letter, which can be followed by any combination of numbers and more lower-case letters.

This is the name of your home directory.

Set the password for the new user account. This is used for log into the desktop and for superuser privileges access.

A good password will contain a mixture of letters, numbers and punctuation and should be changed at regular intervals.

Select the time zone for your ubication. If the desired time zone is not listed, then please go back to the step "Choose language" and select a country that uses the desired time zone (the country where you live or are located).

The two more important options are the following
  • Guided - Use entire disk: This option deletes all existing partitions on the hard drive and creates new ones so that the computer only has one operating system.
    Note: If you have any other operating system it will be removed.
  • Manual: With this option you can configure the partitions manually and thus maintain another operating system if it exists. Partitions can be deleted and resized to make room for the new system.
    Warning: Do not delete the NTFS (Windows) or APFS (MacOS) partition types if you want to keep an existing operating system.

In this part of the installation, the most normal thing is to leave the options that appear by default, only if you want to select a different option should you change the default options.
Warning: You should always select the option to use a network mirror. Otherwise you will not have the package manager for system updates, application installation, etc.
Note: On the proxy entry screen it is generally not mandatory to enter any value.

The boot loader is shown when the system boots and you can select some options like run Obision, run other operating system (if exists) or run some utilities.
Note: You must select install the GRUB loader always otherwise the system may not boot.
Note 2: Generally the target disk for install GRUB will be /dev/sda or /dev/nvme0.
