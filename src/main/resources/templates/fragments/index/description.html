A Debian based Linux distribution

Obision is a distribution of a customized Debian 12 installation that uses the Linux kernel and GNOME desktop environment.

It contains some applications and desktop extensions installed and configured for a better user experience.

You can download the original Debian ISO installer if you want to test the original installation and other desktops.

Obision is designed for desktop using GNOME. There are many other desktop alternatives like KDE, XFCE, MATE, etc. all of them very good and included in other Linux distributions.

Obision is developed for computers with AMD or Intel CPUs of 64 bits only.
